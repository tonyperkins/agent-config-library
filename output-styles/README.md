# Output Styles

Output style configs control how Claude Code formats responses. This is an
emerging config type — community consensus on best practices is still forming.

## Placeholder

This directory is reserved for output style templates once the format
stabilizes. For now, see the Claude Code docs for current output style options.

## What output styles can do
- Control response verbosity (concise vs. detailed)
- Enforce code block formatting preferences
- Set default explanation depth
- Customize how diffs are presented
