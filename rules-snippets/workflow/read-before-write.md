## Workflow
- Before modifying any files, restate the goal in your own words and list the files you think are relevant
- Identify the top 3 risks of the task - what's most likely to go wrong?
- Provide a step-by-step plan with verification for each step before writing any code
- Do not modify files until the user confirms your understanding is correct
- If you're unsure about the goal, ask - don't guess and start coding
