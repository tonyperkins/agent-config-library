## Dependencies
- Before importing a package, verify it is actually installed: `npm list <package>` or `pip show <package>`
- Never assume a package exists based on training data - always verify at runtime
- When using a framework API (React, Next.js, Django, etc.), check the installed version's docs, not what you remember
- If you need current API docs, use Context7 MCP or fetch the official docs - don't rely on potentially outdated knowledge
- If a package is not installed, ask before installing it - do not auto-install
